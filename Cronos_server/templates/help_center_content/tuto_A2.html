A quick tour of the application

Cronos-Connect

This is the login page. Enter your username and password and click on login

Once connected, you'll see the application's main page and a Connected success message Then, we have two possible scenarios :

If your application has needed to take any action on your crons, you'll see the corresponding logs appear on the page

If no action is required, the application will tell you clearly

To make things clearer, you can click the Clear logs button and your home page will be emptied of all messages

The Refresh your cron button will allow the application to check if any actions need to be taken

If any action is taken, the corresponding logs will appear. If no action is required, the message Nothing to fetch will inform you of that

If you feel you have finished, click on "Logout"

You will be returned to the login page
