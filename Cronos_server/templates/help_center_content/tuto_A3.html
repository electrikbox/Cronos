How to create a cron

To create a cron, you need to specify several fields in the creation form to define the time, month, day of the month, day of the week and command to be executed. For more detailed command specifications, please refer to the tutorials below.

The required fields are as follows :


  • Time : In this field, you can enter the exact time (to the minute) when you want your cron job to run
  • Day : Here, it represents "Day of the month". For example, if you select "11", then the month of January, your command will run every January 11
  • Month: This field specifies the month of the year in which the task is to be executed
  • Week's day : This field allows you to specify the day of the week when the command will be executed


The astériks (*) is a wildcard character that means "each / every" Take a look at the examples above to understand how to use it :


  • ① : The asterisk in "Week's day" means that your command will run every January 11th, regardless of which day of the week it falls on

  • ② : The asterisk in "Day" means that your command will run every day of the month, regardless of the month or day of the week (here, your command will run every friday of january, regardless of the day of the month it falls on)

  • ③ : The asterik in "Month" and "Day" means that your command will run every friday (for this example) of the month, regardless of the month or the day of the week
