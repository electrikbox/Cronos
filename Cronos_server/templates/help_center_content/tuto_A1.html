How to install the application on my computer ?

For Cronos to work properly, you need to download and install our Cronos-Connect application. Here's how to do it :


First, go to the Downloads page to find the version that matches your computer's system. We recommend that you open this page in a new browser tab, so that you can continue following the next steps in this tutorial.


Then click on the appropriate download link, and the download will start instantly


For macOS

--WIP-- Once the download is complete, open the file and drag the application to the Applications folder


For Linux

--WIP-- Once the download is complete, open the file and drag the application to the Applications folder


Once the application has been installed, all you need to do is log in using the same identifiers as for logging in to the present site (username + corresponding password)
