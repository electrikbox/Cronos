How to delete a cron

If you delete your cron, it will completely disappear from your dashboard and you'll have to recreate it from scratch if you want to run the same command again with the same options.

To delete a cron, simply click on the trash icon on the right-hand side of your dashboard. A confirmation message will appear and : if you're sure you want to permanently delete this cron, you can click OK. Your cron will then be deleted
